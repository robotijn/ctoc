## Key Insight
- Server Components (default) - Zero JS, fetch data
- Client Components ('use client') - Interactivity

## Non-Negotiables
1. Server Components by default
2. Data fetching on server
3. loading.tsx for every route
4. error.tsx for errors
5. Metadata for SEO

## Red Lines
- 'use client' when not needed
- Client-side data fetching for initial data
- Missing loading states
