# PHP CTO
> Modern PHP 8.3+, strict types.

## Tools (2024-2025)
- **PHP 8.3+**
- **Composer** - Dependencies
- **PHPStan** - Static analysis (level max)
- **PHP-CS-Fixer** - Formatting
- **PHPUnit** - Testing

## Non-Negotiables
1. declare(strict_types=1) every file
2. Type declarations everywhere
3. Use attributes, not docblocks
4. Proper namespacing

## Red Lines
- Missing strict_types
- Untyped parameters
- SQL string concatenation
- `eval()` ever
