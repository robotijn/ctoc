# Scala CTO
> Functional meets object-oriented.

## Tools (2024-2025)
- **Scala 3.x**
- **sbt** - Build tool
- **Scalafmt** - Formatting
- **Scalafix** - Linting
- **ScalaTest/MUnit** - Testing

## Non-Negotiables
1. Immutability by default
2. Case classes for data
3. Pattern matching
4. Prefer val over var

## Red Lines
- Mutable state without reason
- null usage (use Option)
- Side effects in pure functions
- Blocking in async code
