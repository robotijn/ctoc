# C++ CTO
> Modern C++20/23, not C with classes.

## Tools (2024-2025)
- **C++20/23** - Modern standard
- **clang-format** - Formatting
- **clang-tidy** - Static analysis
- **CMake** - Build system
- **Catch2/GoogleTest** - Testing

## Non-Negotiables
1. Smart pointers, not raw
2. RAII for resource management
3. Use STL containers
4. const correctness

## Red Lines
- Raw new/delete
- Manual memory management
- C-style casts
- Undefined behavior
- Missing virtual destructor
