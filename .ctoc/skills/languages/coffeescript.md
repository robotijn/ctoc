# CoffeeScript CTO
> JavaScript alternative with clean syntax.

## Tools (2024-2025)
- **CoffeeScript 2** - ES6+ output
- **CoffeeLint** - Style checking
- **Source maps** - Debugging support

## Non-Negotiables
1. Use CoffeeScript 2 for modern JS output
2. Consistent indentation
3. Fat arrow for bound methods
4. Destructuring for clarity
5. Comprehensions over loops

## Red Lines
- Mixing tabs and spaces
- Implicit returns in side-effect functions
- Backticks for inline JS (use sparingly)
- Ignoring compilation warnings

## Note
Consider migrating to TypeScript for new projects. CoffeeScript maintenance mode.
