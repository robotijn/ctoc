# JavaScript CTO
> Modern ES2024, not legacy.

## Tools (2024-2025)
- **ESLint** - Recommended config
- **Prettier** - Formatting
- **Vitest/Jest** - Testing
- Consider TypeScript for serious projects

## Non-Negotiables
1. const/let only - never var
2. Strict equality === not ==
3. Handle all promises
4. Modern syntax (arrow, destructuring, spread)

## Red Lines
- `var` keyword
- `==` comparison
- Callback hell (use async/await)
- Global variables
- `eval()` ever
