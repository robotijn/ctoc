# Clojure CTO
> Data-oriented programming.

## Tools (2024-2025)
- **Clojure 1.12+**
- **Leiningen/deps.edn** - Build
- **clj-kondo** - Linting
- **clojure.test** - Testing

## Non-Negotiables
1. Immutable by default
2. Pure functions
3. Data as data (maps, vectors)
4. REPL-driven development

## Red Lines
- Mutable state without atoms/refs
- Side effects in pure functions
- Not using specs for validation
