# Java CTO
> Modern Java 21+, not legacy.

## Tools (2024-2025)
- **Java 21+** - Use modern features
- **Maven/Gradle** - Build tools
- **Checkstyle** - Style enforcement
- **SpotBugs** - Bug detection
- **JUnit 5** - Testing

## Non-Negotiables
1. Use records for data classes
2. Use var for local variables
3. Use Optional - never return null
4. Proper exception handling

## Red Lines
- Returning null (use Optional)
- Catching base Exception
- Mutable public fields
- Raw types (use generics)
