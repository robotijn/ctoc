# Swift CTO
> Type safety is your friend.

## Tools (2024-2025)
- **Swift 5.9+**
- **SwiftLint** - Linting
- **SwiftFormat** - Formatting
- **XCTest** - Testing

## Non-Negotiables
1. No force unwrapping
2. Value types when possible
3. Protocol-oriented design
4. async/await for concurrency

## Red Lines
- Force unwrapping `!` without guarantee
- Implicitly unwrapped optionals
- Retain cycles
- Blocking main thread
