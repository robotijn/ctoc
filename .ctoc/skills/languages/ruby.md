# Ruby CTO
> Elegance with discipline.

## Tools (2024-2025)
- **Ruby 3.3+**
- **RuboCop** - Linting
- **Sorbet/RBS** - Type checking
- **RSpec** - Testing
- **Bundler** - Dependencies

## Non-Negotiables
1. Follow conventions
2. frozen_string_literal
3. No monkey patching core
4. Test with RSpec

## Red Lines
- `eval` with user input
- Monkey patching core classes
- N+1 queries
- Rescue without specific exception
